Natural Language Processing

Text files are pre-processed, segmented, and annotated using spaCy library for Named Entity Recognition (NER) and Parts of Speech (POS) tagging. NER identifies entities like people, organizations, and locations, while POS tagging assigns grammatical categories to words. spaCy, an open-source NLP tool, is utilized for these tasks, resulting in annotated text files with marked entities and parts of speech.
